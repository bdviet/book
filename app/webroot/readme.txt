Bài học: Đưa giao diện vào chickenrainshop - phần 1

Thư mục gồm có:
*Thư mục:
  - Thư mục css: chứa các file css, đặt các file trong thư mục này vào trong thư mục webroot/css của project.
  - Thư mục fonts: đặt cả thư mục này vào trong thư mục webroot của project.
  - Thư mục js: chứa các file js, đặt các file trong thư mục này vào trong webroot/js của project.
*File: Gồm các file giao diện mới cho chickenrainshop"
  - home.html: trang chủ
  - category.html: xem chi tiết sách trong danh mục (view category)
  - writer.html: xem chi tiết sách của một tác giả (view writer)
  - search.html: tìm kiếm sách
  - book.html: xem chi tiết sách (view book)
  - File ban-quyen.txt
=> Trong bài học này hiện tại chỉ sử dụng file home.html và các file trong các thư mục css, js, fonts để chỉnh sửa lại layout default của project. 
Các file còn lại sẽ sử dụng cho các bài học tiếp theo (sẽ update sau nếu có).
